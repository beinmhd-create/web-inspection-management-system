# web inspection management system

A web-based inspection management system developed to digitize workplace safety inspections and streamline reporting processes.

Overview

This project was created to improve inspection efficiency by replacing manual inspection forms with a centralized digital workflow.

Features
Multi-category inspection forms
First Aid Kit Inspection
APAR Inspection
Safety Helmet Inspection
Spill Kit Inspection
Safety Sign Inspection
Report History Management
CSV Export
DOC Export
Progress Tracking Dashboard
Local Storage Data Persistence
