
A web-based safety inspection management system designed to digitize HSE inspection processes and improve reporting efficiency.

## Features
- First Aid Kit Inspection
- APAR Inspection
- Spill Kit Inspection
- Safety Helmet Inspection
- Safety Sign Inspection
- Inspection History Management
- CSV Export
- DOC Export
- Progress Tracking Dashboard
- Local Storage Data Persistence
